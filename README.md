# PanIndex
[![Deploy](https://www.herokucdn.com/deploy/button.png)](https://heroku.com/deploy?template=https://github.com/pigeonpig/PanIndex)

## 简单的网盘目录列表
- [x] 跨平台、易部署
- [x] 多模式，多账户
- [x] 多主题
- [x] 下载直链
- [x] 防盗链

## 目前支持的网盘
- 本地目录
- 天翼云网盘
- teambition盘（个人、项目、国际服）
- 阿里云盘
- OneDrive国际版
